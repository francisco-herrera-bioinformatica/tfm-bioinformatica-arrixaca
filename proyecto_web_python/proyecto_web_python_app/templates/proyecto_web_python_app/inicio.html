Trabajo Fin de Máster

Sistema de recogida y procesamiento de datos

Introducción

Este sistema se ha desarrollado a partir de los datos proporcionados por el Laboratorio de Terapia Celular y Tisular del Servicio de Hematología del Hospital Virgen de la Arrixaca.

El objetivo es realizar un primer punto de contacto con el problema que actualmente padecen los trabajadores de dicho departamento, proporcionando un estudio exhaustivo y un prototipo que pueda constituir una posible solución al mismo.

Donaciones

Subir documentos asociados a una donación (nueva o ya existente), haciendo hincapié en los principios de escalabilidad y heterogeneidad propios del entorno de trabajo.

Consultas

Consultar información almacenada en el sistema de almacenamiento de datos para su extracción o comparación.

Documentos

Visor de documentos almacenados en el gestor documental para comprobar su existencia.
